CLAMP

The original idea with the clamp was to grab several balls at a time. Then, since we only had 2 motors left, the clamp had to be designed without a motorisation to grab the ball. The 2 motors were used to control the height of the clamp and the angle of the arm.

We ended up with this clamp (picture below). Elastic strings were used so that applying force onto the ball would trap them inside the net. The main advantages are that the balls do not need to be in a particular place to be grabbed, hence removing a lot of indeterminism.
