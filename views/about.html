Battling through the COVID pandemic

The COVID-19 crises made us work from home during the whole project. We adapted to that unforseen constraint and build ourselves remote setup to work as a group.

After Jean-Baptiste was done constructing and flashing the robot, Nicolas collected it and set up an SSH server on it so that the entire team could connect to it through Nicolas's IP via port-forwarding.

So that everyone could run test and have visual on the robot, Nicolas also set up a live video stream on YouTube and Google Meet. That was especially useful to Brehima for his movement tests.
